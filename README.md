KiCad
==============================
Docker上でKiCadを動かす．

BUILD
--------------------
```
$ git clone https://github.com/HeavyMoon/KiCad.git
$ cd KiCad/
$ docker build -t kicad .
```

HOW TO USE
--------------------
```
$ ./kicad.docker
```
